## Build

### Bash

```sh
./goyek.sh -h
./goyek.sh
```

### PowerShell

```pwsh
.\goyek.ps1 -h
.\goyek.ps1
```

### Visual Studio Code

`F1` → `Tasks: Run Build Task (Ctrl+Shift+B or ⇧⌘B)`
